CSS Animation

Transitions

Used to change css property smoothly over a duration.

{
  transition: property duration easing-function delay;
}
<style>
  .box {
    height: 50px;
    width: 50px;
    background: red;
    transition: transform 1s;
  }
  .box:active {
    transform: translateX(100px);
  }
</style>
<div class="box"></div>

Transition Options

transition-property

Used to specify the css property on which the transition is applied

{
  transition-property: all | height | width | transform | css_property;
}
<style>
  .box {
    height: 50px;
    width: 50px;
    background: red;
    transition: height 1s;
  }
  .box:active {
    height: 100px;
  }
</style>
<div class="box"></div>

Transition Options (Contd...)

transition-timing-function

Used to specify the speed behaviour of the animation

{
  transition-timing-function: linear | ease | ease-in | ease-out | ease-in-out | step-start | step-end | steps(int,start|end) | cubic-bezier(n,n,n,n) | initial | inherit;
}
cubic-bezier 
{
  transition-timing-function: cubic-bezier(p0,p1,p2,p3);
}

Used to defined the timing behaviour of the transition in format of a cubic-bezier curve.
P0, P3 being fixed to (0,0) and (1,1) in the curve, p1, p2 being the control points

  50% {transform: translateX(50px);}

animation Options

animation-name

#box {
  animation-name: <animation-name>
}

animation-duration

time for which animation has to run(in seconds)

#box {
  animation-duration: <animation-duration>S
}

animation-timing-function

Used to specify the speed behaviour of the animation same as the one in transitions

{
  animation-timing-function: linear | ease | ease-in | ease-out | ease-in-out | step-start | step-end | steps(int,start|end) | cubic-bezier(n,n,n,n) | initial | inherit;
}

animation Options (contd...)

animation-delay

number of seconds after which the animation has to start

#box {
  animation-delay: <animation-delay>S
}

animation-iteration-count

Number of times the animation has to repeat

#box {
  animation-iteration-count: number|infinite|initial|inherit;
}

animation-direction

specifies the flow of animation, i.e. forward, reverese, alternate etc.

#box {
  animation-direction: normal|reverse|alternate|alternate-reverse|initial|inherit;
}

animation Options (contd...)

animation-fill-mode

Specifies which frame style should the element persist after the animation is executed

#box {
  animation-fill-mode: none|forwards|backwards|both|initial|inherit;
}
none
forwards
backwards
both
@keyframes noneFillMode {
  from {transform: translateX(50px);background: red;}
  to {transform: translateX(200px);background: blue;}
}
#box2 {
  width: 50px; height: 50px; background: gray;
  animation: noneFillMode 5s 2s;
}

animation-play-state

Generally used to pause or resume animation through javascript.

#box {
  animation-play-state: paused|running|initial|inherit;
}

2D Transforms

Transforms

Used to change position, size and shape of an element

{
  transform: translate | rotate | scale | skew | matrix;
}

Transform Options

translate

Used to change position of an element

{
  transform: translate(xpx, ypx) | translateX(xpx) | translateY(ypx);
}
<style>
  .box6 {
    width: 50px; height: 50px;
    background: red;
    margin: 20px;
  }
  .transformC {
    transform: translate(10px, 10px);
  }
</style>
<div class="box6"></div>
<div class="box6 transformC"></div>

Transform Options (Contd...)

matrix

The matrix() method take six parameters, which allows you to rotate, scale, move (translate), and skew elements.

{
  transform: matrix(scaleX(),skewY(),skewX(),scaleY(),translateX(),translateY());
}
<style>
  .transformM:active {
    transform: matrix(2, 0, 0.5, 1, 150, 0);
  }
</style>
<div class="box6"></div>
<div class="box6 transformM"></div>
